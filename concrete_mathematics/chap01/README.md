第1章 递归
==========================

在本章中介绍了三个递归的实例，通过这些实例，可以很好地说明递归的特点。这三个递归问题有以下的共同特点：1、都是采用了数学的方法来研究；2、解决问题的方法都用到了递归。这三个问题，更大数量级的问题的解决办法，都需要依赖小规模问题的解决。
